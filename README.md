## 本地开发

node 需要大于`18`
`cd honeyed-words-generator && yarn install`

`yarn dev`

Let's rock it~

### ubuntu

node 需要大于`18`,我使用的是`22.16.0`

`npm install`

`npm run dev`

注意： 现在可以正常运行了，最好不要再修改 package.json 等依赖文件了
